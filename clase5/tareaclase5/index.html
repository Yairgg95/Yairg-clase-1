Abreviaciones

La etiqueta <abbr> define una abreviación o un acrónimo

Ejemplo


La <abbr title="Organización de las Naciones Unidad">ONU</abbr> fue fundada en 1945.

Salto de linea

La etiqueta <br> inserta un salto de línea, es una etiqueta vacia lo que significa que no tiene una etiqueta final es muy útil para direcciones o poemas

Ejemplo


Aquí

<p>Mis pasos en esta calle<br>
Resuenan<br>
En otra calle<br>
Donde<br>
Oigo mis pasos<br>
Pasar en esta calle<br>
Donde,<br>
sólo es real la niebla.</p>

-Octavio Paz

Enfatizar texto

La etiqueta <em> se utiliza para definir texto enfatizado. El contenido interior normalmente se muestra en cursiva.

Ejemplo


<p>Los <em>esfuerzos</em> siempre dan sus frutos.<p>

División

La etiqueta <div> define una división o sección en un documento HTML.

Ejemplo


<div>
<h2>Este es un heading en una etiqueta div </h2>
<p>Este es un texto en una etiqueta div</p>
</div>

Articulo

La etiqueta <article> especifica contenido independiente y autónomo.

Ejemplo


<article>
<h2>Google Chrome</h2>
<p>Google Chrome es un navegador web desarrollado por Google, lanzado en 2008. ¡Chrome es el navegador web más popular del mundo en la actualidad!</p>
</article>

Hipervínculo

La etiqueta <a> define un hipervínculo, que se utiliza para vincular de una página a otra.

Ejemplo


<a href="https://www.w3schools.com">Visita W3Schools.com</a>

Texto en negrita

La etiqueta <b> especifica texto en negrita.

Ejemplo


<p>Esto es texto normal <b>y esto es texto en negrita</b></p>

Cuerpo del documento

La etiqueta <body> define el cuerpo del documento.

Ejemplo


<body>
<h1>Esto es un título</h1>
<p>Esto es un párrafo.</p>
</body>

Imagen

La etiqueta <img> se utiliza para incrustar una imagen en una página HTML.

Ejemplo


<img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600">

Formulario

La etiqueta <form> se utiliza para crear un formulario HTML para la entrada del usuario.

Ejemplo


<form action="/action_page.php" method="get">
<label for="fname">First name:</label>
<input type="text" id="fname" name="fname"><br><br>
<label for="lname">Last name:</label> <input type="text" id="lname" name="lname"><br><br>
<input type="submit" value="Submit">
</form>

Campo de entrada

La etiqueta <input> especifica un campo de entrada donde el usuario puede ingresar datos.

Ejemplo


<input type="text" id="fname" name="fname"><br><br>

Lista ordenada

La etiqueta <ol> define una lista ordenada. Una lista ordenada puede ser numérica o alfabética.

Ejemplo


<ol>
<li>Agua</li>
<li>Refresco</li>
<li>Leche</li>
</ol>

Lista desordenada

La etiqueta <ul> define una lista desordenada con viñetas.

Ejemplo


<ul>
<li>Honda</li>
<li>Toyota</li>
<li>Nissan</li>
</ul>

Definir fila

La etiqueta <tr> define una fila en una tabla HTML.

Ejemplo


<table>

<th>Month</th>
<th>Savings</th>
</tr>
<tr>
<td>January</td>
<td>$100</td>
</tr>
<tr>
<td>February</td>
<td>$80</td>
</tr>
</table>
